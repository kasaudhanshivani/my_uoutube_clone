# YouTube Clone 🎬

A static YouTube homepage clone built using **HTML and CSS only**.

## 🔥 Features
- YouTube-style header and video grid
- Multiple video cards
- Video duration overlay
- Dark mode toggle (CSS only)
- Responsive layout (basic)

## 🛠 Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)
